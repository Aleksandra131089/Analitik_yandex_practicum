На основе данных использования мобильного приложения для продажи продуктов питания проанализировала воронку продаж, а также оценила результаты A/A/B-тестирования. Инструменты: A/B-тестирование, Python, Pandas, Matplotlib, Seaborn, событийная аналитика, продуктовые метрики, Plotly, проверка статистических гипотез, визуализация данных.
